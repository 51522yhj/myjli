# Product

## Register

brand

## Users

主要用户是招聘方、面试官和技术负责人。他们会在较短时间内判断候选人的后端基础、企业项目经验、AI 应用集成能力和工程化交付意识。

## Product Purpose

Myjli 是于昊骏的在线简历与项目作品集。它不是普通简历 PDF 的网页翻版，而是通过项目证据、流程细节、真实截图和动效叙事展示 Java 后端、企业系统、AI 工程化与 Vibe Coding 经验。

## Brand Personality

清澈、精确、有纵深。页面应有梦幻科技感，但文字表达要真实克制，不夸大经历。

## Anti-references

不要做成普通 SaaS 卡片堆叠、模板化深色科技大屏、纯 CSS 装饰门、无内容支撑的夸张动效、泛泛而谈的项目介绍。

## Design Principles

- 先用图像建立记忆点，再用项目细节证明能力。
- 每个项目都给出业务背景、技术栈、流程和个人职责。
- 动效服务于探索节奏，不遮挡阅读。
- 内容必须可面试复述，避免无法证明的夸大表达。
- 真实截图优先于抽象装饰。

## Accessibility & Inclusion

目标满足 WCAG AA 基础要求。必须支持键盘访问、可读对比度、图片 alt、按钮语义和 prefers-reduced-motion 降级。
