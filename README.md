# Myjli

于昊骏的静态在线简历与项目作品集，使用 React + Vite + Tailwind CSS + Framer Motion 构建，面向 Cloudflare Pages 部署。

## 本地运行

```powershell
npm install
npm run dev
```

## 构建

```powershell
npm run build
npm run preview
```

## Cloudflare Pages

- Build command: `npm run build`
- Output directory: `dist`

- 企业项目：智审 aimod、放心签 signAtEase
- 个人项目：YYOJ、录合同智能体 / Spring AI 改造
- VibeCoding：源创潮牌工厂直营店、知时客、Android App、Work Record Desktop、历史长河
